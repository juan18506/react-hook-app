## Installation and usage

### Runing the project

Clone this repository: 

```
git clone https://github.com/juan18506/react-hook-app.git
cd react-hook-app
```

Install packages :

```
pnpm install
```

When installation is complete, run :

```
pnpm dev
```

To run tests:

```
pnpm test
```
